Hexo+Github建站搭建属于你的博客（初识）

发表于 2020-05-10 分类于 hexo 阅读次数：
本文字数： 2.7k 阅读时长 ≈ 2 分钟

想要拥有自己的门户网站？想要提高自我的写作水平？想要获得更多多的成就感？想对世界分享你的idea？想要展示自己技能，创造力和才能？那么拥有一个博客blog(博客)应该会是个不错的选择,接下来就就让我们一起用 Hexo+Github搭建属于你的博客，对你没听错真正属于你的博客！那就让我们开始吧，come on！！！

什么是Hexo

Hexo是一款基于Node.js的快速、简洁且高效的静态博客框架，依赖少易于安装使用，使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。可以方便的生成静态网页托管在GitHub和Coding上，是搭建博客较好的选择。

教程分为三个阶段：

  • 第一阶段：hexo的初级搭建还有部署到github page上，以及个人域名的绑定。
  • 第二阶段：hexo的基本配置，更换主题，实现多终端工作，以及在coding page部署实现国内外分流
  • 第三阶段：hexo添加各种功能，包括搜索的SEO，阅读量统计，访问量统计和评论系统等。

然后创建SSH,一路回车

ssh，简单来讲，就是一个秘钥，其中，id_rsa是你这台电脑的私人秘钥，不能给别人看的，id_rsa.pub是公共秘钥，可以随便给别人看。把这个公钥放在GitHub上，这样当你链接GitHub自己的账户时，它就会根据公钥匹配你的私钥，当能够相互匹配时，才能够顺利的通过git上传你的文件到GitHub上。

这个时候它会告诉你已经生成了.ssh的文件夹。可以在你的电脑找到这个文件把id_raa.pub里面的信息复制，或者git bash中输入
